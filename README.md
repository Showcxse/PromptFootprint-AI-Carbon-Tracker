# PromptFootprint -- See the carbon cost before you send

    This is a personal project / web-app / tool that tracks the carbon output of AI/LLM Prompts.

    I got the idea when one of my lecturers randomly mentioned that AI prompts do create a carbon output. And I have been unfathomably bored because apparently they won't teach me anything relating to coding here before my third term

    All Data is valid through 2026 and is powered by the Electricity Maps API
